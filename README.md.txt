# Football Data Analysis

This project analyzes the performance of football teams and players using Python and Pandas.

## Data

The data is provided in the football_matches.csv file. It contains information about football matches, including team names, player names, goals scored, and match results.

## Analysis

The analysis includes:
- Team performance analysis (goals scored, wins, losses)
- Player performance analysis (goals scored)

## Usage

To run the analysis, execute the analysis.py script:

```bash
python analysis.py
